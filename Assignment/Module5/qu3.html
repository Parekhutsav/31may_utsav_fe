Qu-3 Semantic element in HTML5?

-> Semantic element have meaningful names which tell about the type of content.
-> In HTML5 there are many Semantic elements given below.
Ex:-
(1)header
(2)nav
(2)aside
(3)article
(4)section
(5)footer
->header:- It is used to define a header for a document or a section.
Example: <header></header>
->nav:- It is used to define navigation links related to the current page.
Example: <nav></nav>
->aside:- It is used to specify some related content on the side of the main content.
Example: <aside></aside>
->article:- It represents a some blogs,articles,posts etc.
Example: <article></article>
->section:- It is used to divide content in section.
Example: <section></section>
->footer:- It defines footer content that is after the main content.
Example: <footer></footer>
