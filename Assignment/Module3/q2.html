Q.2 What are tags and attributes in HTML?

Ans:-
1.Tags:-Tag is used to show content to the browser.
->There are two types of tags:-
a.Pair Tag
b.Non-pair Tag
a.Pair Tag:-
Pair tag have a starting tag and ending tag.ending tag start with '/'.
Ex:- <title></title>
b.Non-pair Tag:-
Non-pair tag have only one starting tag.
Ex:- <br>

2.attributes:-Attribute is use to define the property of the element and it is compulsory writen in starting tag.
Ex:- <html lang="en"></html>
