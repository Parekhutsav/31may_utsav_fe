Q.15 How to insert a picture into a background image of a web page? With Example.


Ans:-
-> We insert a picture in background with background-image attribute in body tag but in HTML5 it was depricate.
-> In HTML5 we can insert a background image with css property.It is a "background-iamge" in style tag.
Ex:-
<style>
body{
background-iamge:url;
}
</style>
