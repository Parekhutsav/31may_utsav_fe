Q.10 How can we club two or more rows or columns into a single row or column in an HTML table? With Example.


Ans:-
-> To combine two or more rows or columns into a single row or column,we need to use colspan and rowspan attribute of HTML.
-> rowspan is used to marge two or more rows.
-> colspan is used to marge two or more columns.
Ex:-
<table>
<tr>
<td colspan="2">
abc
</td>
</tr>
<tr>
<td rowspan="2">
xyz
</td>
</tr> </table>
