Q-2 How to Apply CSS Using JQuery, How to Add Class and Remove Class in Jquery, JQuery Animation?

Ans:
->To apply CSS using jQuery, you can use the .css() method which is used to get or set a style property for the selected content.
->To apply CSS using jQuery, you can use the .addClass() method to add a class to an element.
->the .removeClass() method to remove a class from an element.
->The animate() function of jQuery can be used for creating animations on the web page.

[1] Apply CSS
$("#id").css("property","value");
Example :
$("p").css("color", "red"); // Changes all paragraphs color to red

[2] Add/Remove Class
//Adding a class
$(".classname").addClass("newclass");
Example :
$("button").click(function(){
$(this).addClass("active"); // Adding active class to clicked button
});

//Removing a class
$(".classname").removeClass("oldclass");
Example :
$("button").click(function(){
$(this).removeClass("active"); // Removes active class from mouse outed button
});

[3] jQuery animation
$("selector").animate({propertyname:"value"},speed);
Example :
$("#box").click(function(){
$(this).animate({left:'+=150px', opacity:'0.5'}, "slow");
});
