Q-5 What are undeclared and undefined variables?

Ans:
[1.] Undeclared variables
->An undeclared variable is a variable that has not been declared before it is used in the code.
->If we use ‘typeof’ operator to get the value of an undeclared variable, we will face the runtime error with the return value as “undefined”.
Ex:-
console.log(x)
->In this example 'x' variable is not defined anywhere. So it is an undeclared variable.

[2.] undefined variable
->It occurs when a variable has been declared but has not been assigned any value.
Ex:-
var x;
console.log(x); // Output : "undefined"
