Q-12 What are all the types of Pop up boxes available in JavaScript?

Ans:
1. Alert Box - It is a simple pop-up box that displays a message and an OK button to close it.

2. Confirm Box - It is a pop-up box that displays a message and asks for user confirmation before proceeding with an action.

3. Prompt Box - It is a pop-up box that displays a input field and it is given by users.
