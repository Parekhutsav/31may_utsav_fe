Q-9 What are all the looping structures in JavaScript?

Ans:
1. For Loop :
Syntax : for (let i = 0; i <= n; i++) { //code block }

Ex:
  for(let i = 0; i <= 5; i++){
     console.log("The value of i is " + i);
}

2. While Loop
Syntax : while(condition) { //code block }

Example :
let i = 0;
while(i < 5) {
console.log("Hello World");
i++;
}

3. Do...While Loop
Syntax : do { //code block } while(condition);

Example :
let x = 10;
do {
console.log(x);
x--;
}while(x >= 0);
