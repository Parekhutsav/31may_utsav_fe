Q-4 Name a few CSS style components

Ans:-

->CSS style components means properties of the CSS.
->It's are given below:-
*Color properties:-
->There are many properties of the Color properties.
->Like, 'color:for text color', 'background-color:for background color', 'border-color:for border color' etc.
*background properties:-
->There are many properties of the background properties.
->Like, 'background-image:to set image in background', 'background-color:for background color', 'background-size:to define the size of the background' etc.
*Text properties:-
->There are many properties of the text properties.
->Like, 'text-align:to set position of the text(left,right,center)', 'text-decoration:to set decoration of the text(underline)', 'text-transform:to set text in uppercase,lowercase etc' etc.
