Q-11 How to center block elements using CSS1?

Ans:-

->We can center block level element with margin attribute of CSS1.
->We have to give 'auto' value of the margin to set the block element in center.
->Ex:-

<style>
  div
  {
    height:200px;
    width:50%;
    border:1px solid black;
    background:darkorange;
    margin:auto;
  }
</style>
