Q-1 What are the benefits of using CSS?

Ans:-

->Using CSS we can apply style in HTML pages and change it's layout.
->We can also manage CSS in external file and linkd this file into HTML file and reduce the length of the HTML file.
->We can write CSS once and then reuse the same sheet in multiple HTML pages.
->We can also manage layout control with 'position', 'float','flex','grid'.
->We can also give animation to the webpages.
->We can also make responsive webpages using 'media' queries.
->We can also control text and paragraphs properties.
