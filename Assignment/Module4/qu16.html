Q-16 What are the advantages and disadvantages of using external style sheets?

Ans:-

*Advantages:-
->External style sheets containe all the CSS of the website in a one file and make easier to read all the CSS.
->It is easier to maintaine the file.
->We can change into the CSS file the reflection of the changes in all the files where the links are provided.
->External file of the css can be linked in multiple HTML pages.
*Disadvantages
->We have to give the link of the external CSS in all the HTML file.
-> It's create a more time to load the website when it's rendering.
