Q-17 What is the meaning of the CSS selector?

Ans:-

->CSS selectors are used to select the content you want to style.
->-It is select according to it's id,class,element name etc.
-> There are some types of the selectors are given below:-
  (1.)Element Selector
  (2.)Id Selector
  (3.)Class Selector
  (4.)Universal Selector
  (5.)Group Selector
  (6.)Pseudo-Element Selector
  (7.)Pseudo-Class Selector
->Ex:-
   <style>
   .abc{background:red;} ('class selector')
   #abc{background:red;} ('id selector')
   div{background:red;} ('element selector')
   *{background:red;} ('Universal selector')
   .abc,p{background:red;color:black} ('Group selector')
   </style>
