Q-18 What are the media types allowed by CSS?

Ans:-

->Media type alloes mainly 2 types of the media:-
[1.] screen:-
->This media type is used for computer screen, tablets, smartphone and similar devices.
->Ex:-
@media screen and (max-width:700px){
background:red;
}
[2.] print
->We have to define CSS in this type of media when the content is printed.
->Ex:-
@media print{
p{color:black;}
}
