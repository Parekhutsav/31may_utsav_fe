Q-3 What is the difference between CSS2 and CSS3?

Ans:-

*CSS2:-
->CSS2 is a updated version of the CSS1.
->CSS2 is used basic selector like element selector,class selector and ID selector.
->In CSS2 we used 'float' and 'position' properties for the layout control.
->CSS2 provides basic properties of background like 'background-color' and 'background-image' etc.
->CSS2 is not support media queries for responsive design.
->CSS2 is not support animation properties.
*CSS3:-
->CSS3 is a latest and currently usable version of the CSS.
->CSS3 is used updated selector like pseudo-class,pseudo-elements etc.
->In CSS3 we used some new properties like 'flex' and 'grid' for the layout control.
->CSS3 provides some new properties of background like 'gradients' etc.
->CSS3 is support media queries for responsive design.
->CSS3 is support animation properties.
