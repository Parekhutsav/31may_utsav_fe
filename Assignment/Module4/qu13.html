Q-13 What are the ways to integrate CSS as a web page?

Ans:-

->There are mainly three ways to integrate CSS as a web page.
   1. Inline CSS
   2. Internal CSS
   3. External CSS

[1.] Inline CSS:-
->Inline CSS is an unique way to give CSS for a single element.
->Inline CSS is define in starting tag of an element.
->In starting tag we have to define 'style' attribute for CSS and apply all the proprties of the CSS.
->Ex:-
<html>
 <body>
 <p style="color:blue">
  Lorem ipsum dolor sit amet consectetur adipisicing elit. Eius reiciendis amet nihil odio quo autem sit veritatis nam, fugiat, nisi iste officiis doloribus, in porro nobis velit. Pariatur, sunt. Alias.
 </p>
 </body>
</html>

[2.] Internal CSS:-
->Internal CSS is define in 'style' tag and it's define into head tag.
->In Internal CSS apply the CSS with name of the element or clsss name and id name.
->Ex:-
<html>
 <head>
  <style>   
  p{color:blue}
  </style>
 <body>
 <p>
  Lorem ipsum dolor sit amet consectetur adipisicing elit. Eius reiciendis amet nihil odio quo autem sit veritatis nam, fugiat, nisi iste officiis doloribus, in porro nobis velit. Pariatur, sunt. Alias.
 </p>
 </body>
 </head>
</html>

[3.] External CSS:-
->External style sheet is used for give CSS to the webpage with external file of CSS.
->We have to create external CSS file with '.css' extension , like 'style.css'.
->We have to give external CSS path in webpage with 'link' tag into 'head' tag.
->Ex:-
<head>
   <link rel="stylesheet" href="style.css">
</head>
