Q-15 What are the external style sheets?

Ans:-

->It is a one type of the CSS.
->External style sheet is used for give CSS to the webpage with external file of CSS.
->We have to create external CSS file with '.css' extension , like 'style.css'.
->We have to give external CSS path in webpage with 'link' tag in 'head' tag.
->We have used this external CSS file across multiple webpage.
->Ex:-
<head>
   <link rel="stylesheet" href="style.css">
</head>
