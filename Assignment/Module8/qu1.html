Q-1 What are the advantages of Bootstrap?

Ans:-
-> bootstrap is a front-end framwork that helps to create business related website.
-> It provides predefined classes for easy and quick development of websites.
-> It has responsive design capabilities which makes your site look good on all devices, like: descktop, tablet, phone etc.
-> It has built in support for typography, colors, buttons, navigation, forms, tables etc., so you can just use the right class and markup to have a professional looking web page with minimal effort.
