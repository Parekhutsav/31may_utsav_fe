Q-3 What are the default Bootstrap text settings?

Ans:-
-> Bootstrap include some text settings are given below:
1. Font Family - Open Sans
2. Font Weight - Normal (400)
3. Line Height - 1.5em
4. Letter Spacing - .075rem
5. White Space - Nowrap
6. Font Size - 1rem(16px)
7. Text Alignment - text-left
8. Text Color - #000
9. Font Style - font-style:normal
